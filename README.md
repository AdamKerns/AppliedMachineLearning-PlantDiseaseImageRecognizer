# Plant Disease Image Recognizer

**By Adam Kerns**  
*Derived from [Advancing plant leaf disease detection integrating machine learning and deep learning](https://doi.org/10.1038/s41598-024-72197-2)*

---

## Description

Traditional methods of diagnosing plant diseases rely heavily on expert knowledge and manual inspection, making them inefficient and error-prone. Machine learning offers a scalable solution for early and accurate disease detection in agricultural and commercial farming environments.

## How to Use

Choose one of the following methods to get the project locally:

### Option 1: Clone the Repository

```bash
git clone https://github.com/AdamKerns/AppliedMachineLearning-PlantDiseaseImageRecognizer.git plant_disease_identifier
```
### Option 2: Download as ZIP

1. Click the **Code** button
2. Select **Download ZIP**
3. Extract the files to your desired location

### *Important*

Once the datasets are downloaded and the files are all local, put the root folders of the datasets into the data folder
